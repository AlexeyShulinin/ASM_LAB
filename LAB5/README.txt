For start:
1. compile
2. Run in dosbox and pass as command line parameters: 1. Path to the file(ex: c:\file.txt), 2. Word to search, 3. New word to replace with the word to search
Features:
1. The possibility of replacing a word of longer length with a word of shorter length, a word of shorter length with a word of longer length
2.Works with files larger than 64kb (tests attached)
3.To see the result, you need to open newfile.txt (sometimes you need to use the view in the total commander)
4.Programm works with buffer which helps process data files and buffer size 51 bytes
